## 주요기능 

#### 1. 내 위치(지도)
##### - 해당 페이지에 들어가면 현재 위치가 나오고 카테고리 지역 년,월,일d을 입력하면 해당되는 게시글만 나오는 기능 

#### 2. 채팅
##### - 파티 주최자 및 컨텐츠 참여자들이 대화를 나눌 수 있는 실시간 채팅 구현

#### 3. 검색
##### - 검색어 입력 후 검색어에 제목, 개최 장소, 작성자가 포함되어 있으면 결과로 출력하는 기능

#### 4. ADMIN(관리자 페이지)
##### - 회원관리, 파티 및 축제 참여 통계, 공지사항

#### 5. 예약
##### - 게시글 상세화면에서 참여하기를 누르면 예약되는 시스템(예약 되어있으면 두번 참여가 안됨)
